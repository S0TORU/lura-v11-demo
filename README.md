# LURA V11 Demo

Interactive demo of the LURA Virtual Wet Lab interface (V11).

## Live Demo

Once deployed to GitHub Pages, this will be accessible at:
`https://[your-username].github.io/[repo-name]/`

## Features

- **Landing Page**: Overview of LURA's capabilities
- **Virtual Wet Lab Interface**: Interactive drug screening simulation
- **3D Tumor Microenvironment**: Visualize spatial drug response
- **Real-time Results**: Mock data streaming and visualization
- **Batch Screening**: Multi-compound analysis interface

## Local Testing

Open `index.html` in a browser to test locally.

## Files

- `index.html` - Main application
- `styles.css` - All styling
- `script.js` - Application logic and mock data

## Notes

This is a **mock interface** with simulated data for demonstration purposes.
Backend integration is separate and not included in this demo.
